Tetris Game Project | Front-End Application
A modern, interactive Tetris game implemented with HTML, CSS, and JavaScript. The project focuses on gameplay mechanics, user experience, and responsive design.
Main Features:

-User authentication (Sign in / Sign up) with local storage
-Dynamic Tetris gameplay with piece rotation and movement
-Score tracking and high score management
-Pause and restart functionality
-Responsive and visually appealing interface with animated gradients and effects

Key Focus:

-Creating a scalable and interactive browser-based game
-Writing clean, modular, and maintainable JavaScript code
-Prioritizing gameplay experience with smooth animations and controls
-Implementing dynamic user feedback (score, game over, current player)

Technologies: HTML5, CSS3, JavaScript, Local Storage, DOM Manipulation, Responsive Design
